# Data-Cleaning-and-Preprocessing-Pipeline

This project demonstrates a professional, end-to-end data preprocessing workflow designed to transform raw, unstructured data into a clean, reliable, and model-ready dataset. In real-world scenarios, data is rarely perfect. It often contains:

 - Missing values
- Inconsistent formats
- Outliers
- Noise
  
# 🎯 Objectives
- Deliver a structured data cleaning process
- Build reusable preprocessing pipelines
- Ensure data consistency, quality, and reproducibility
# 🛠️ Tools:

- Chat GPT
- Jupyter Notebooks
- Google Colab
- Pandas
